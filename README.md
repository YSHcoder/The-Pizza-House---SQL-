🍕 SQL Pizza Sales Analysis
📌 Project Overview

The SQL Pizza Sales Analysis project focuses on analyzing pizza sales data using SQL to extract meaningful business insights. This project demonstrates the use of SQL for data exploration, aggregation, joins, grouping, sorting, and advanced analytical queries to support data-driven decision-making.

🎯 Objectives
Analyze customer ordering patterns.
Identify top-selling pizzas and categories.
Calculate total revenue and sales performance.
Understand order distribution by size and category.
Generate business insights using SQL queries.
🛠️ Technologies Used
SQL (MySQL)
MySQL Workbench
📂 Dataset

The dataset contains information about:

Orders
Order Details
Pizzas
Pizza Types

These tables are connected using primary and foreign keys to perform relational data analysis.

📊 Key Analyses Performed
Basic Analysis
Retrieved the total number of orders placed.
Calculated total revenue generated from pizza sales.
Identified the highest-priced pizza.
Determined the most commonly ordered pizza size.
Listed the top 5 most ordered pizza types.
Intermediate Analysis
Calculated total quantity sold for each pizza category.
Analyzed order distribution by hour of the day.
Determined category-wise pizza distribution.
Calculated the average number of pizzas ordered per day.
Identified the top 3 pizza types based on revenue.
Advanced Analysis
Calculated the percentage contribution of each pizza category to total revenue.
Analyzed cumulative revenue generated over time.
Identified the top 3 highest revenue-generating pizzas within each category.
Applied advanced SQL concepts including CTEs, window functions, aggregate functions, subqueries, and multi-table joins.

SQL Query Writing

GROUP BY & ORDER BY

Data Analysis
Business Insight Generation
🚀 Project Outcome

This project demonstrates how SQL can be used to transform raw transactional data into actionable business insights. By analyzing pizza sales trends, customer preferences, and revenue performance, the project showcases practical SQL skills that are widely used in data analytics and business intelligence.
